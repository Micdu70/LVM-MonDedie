# Script d'installation & gestion LVM


Note: Une partition /home séparé est indispensable !

## Utilisation:
````
cd /tmp
git clone https://github.com/exrat/LVM-MonDedie
cd LVM-MonDedie
chmod a+x lvm-mondedie.sh && ./lvm-mondedie.sh
````
